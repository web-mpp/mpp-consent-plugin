GDPR/ePrivacy/CCPA/PIPEDA compliant cookie consent manager with Google Consent Mode v2.

== Description ==

WP Cookie Shield is a comprehensive cookie consent solution for WordPress that helps you comply with GDPR, ePrivacy, CCPA, PIPEDA, and Law 25 requirements.

= Key Features =

* Cookie consent banner (top or bottom) with Accept All, Reject, and Preferences options
* Granular preferences modal with per-category toggles
* Google Consent Mode v2 integration (all 8 consent parameters)
* Script blocking until consent is granted
* Cookie scanner to auto-detect cookies from installed plugins and theme
* Consent audit log for compliance records
* WP Consent API compatible
* Geolocation support — show banner only to visitors in regulated jurisdictions
* Do Not Track and Global Privacy Control support
* Shortcodes for cookie tables and preference links

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-cookie-shield/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Cookie Shield to configure
